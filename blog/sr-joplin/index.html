Software Review: Joplin

Posted on 18 Feb 2024

Joplin is a popular open-source note-taking application. I have been using Joplin for over a year now and here is my review:

Good desktop client, used the Snap package on Debian and everything just works. Android app is a little ugly and slow.

Some features I wish it had:

  • The ability to edit existing files. I can’t edit my markdown files for this blog.
  • Journalling is a common usecase for note-taking apps so a dedicated button to make a new note with the current date.

The features it does have are perfectly implemented. I would give it a 4/5 :)
